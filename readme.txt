== Description ==

The Vina Custom HTML Widget is a small plugin for Wordpress. Its allows you to create your own HTML Widget using a WYSIWYG editor. And this widget support many skins. You can use this plugin to display HTML Text, Quotes, Jokes, Tips etc. The information can be stylized and displayed many different ways.

Features:

- Support multi Wordpress Version.
- Customize User CSS formatting.
- Allow loading wordpress widget inside post, page.
- 6 skins support.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE7+, Firefox 2+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation
- Easy to use.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'vina-text-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Vina Custom HTML Widget' to your sidebar
